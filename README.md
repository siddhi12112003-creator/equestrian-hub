# 🐴 Equestrian Hub - Horse Riding Academy Management System

## About
A full-stack web application to manage a Horse Riding Academy.
Built as a personal project inspired by my experience as Horse Riding Captain.

## 🔗 Live Demo
https://equestrian-hub--siddhi12112003.replit.app
Login: admin / admin123

## Features
- 🐎 Horse Registry - Add, update, delete horses
- 🧑 Rider Profiles - Manage riders with skill levels
- 📅 Session Scheduler - Book riding sessions
- ✅ Attendance Tracker - Mark present/absent
- 🔐 Authentication - Login/Logout system
- 📊 Dashboard - Real-time stats

## Tech Stack
- Frontend: React.js + TypeScript + Tailwind CSS
- Backend: Node.js + Express.js
- Database: PostgreSQL
- Deployment: Replit
